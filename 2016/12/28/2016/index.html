我的2016年

2016.12.28 sherryriver 年终总结  pv:

上图防水 github 时间线200+达成！！！(同志仍需努力啊)

又到提笔年终总结时候了，15年那篇流水账还热乎着呢。好多买的书没有看完，好多新东西没有学会，16年就要走了。

手机刷着回家火车票，大脑刷着这一年零零总总…
大年30某宝的集五福开奖 人品爆棚分了一杯羹 拉开我16年的帷幕。

如果能用一个字形容我的16年的工作 我选择 “变“
离开传统it行业投入互联网公司怀抱。没有什么是一帆风顺，缺乏相关经验导致面试受挫，回来恶补知识。传统软件公司抛来橄榄枝，忍痛拒绝。过程是曲折的，但品尝丰收的果实是喜悦的。如愿的接触到自己想要的东西，第一次以技术者的身份参与双十一还有后面的双十二，虽然累成狗但是大呼过瘾，这些没有公司平台给予支撑 都是纸上谈兵。而且感恩遇到好的“引路人”，一个有重度代码洁癖的cto 跟着他学会了很多，也深刻的知道自己想成为一个以技术为标榜的人。

一个字形容我的16年的个人职业发展 我选择 “专“
不再浮躁，告别做一个全栈开发geek的假大空想法。踏实的专注后端服务开发，前端作为自己的业余兴趣为指导方针 几年内不动摇。屁股后面没有需求催了，前端就成了自己的个人爱好 所以个人博客弄的红红火火恍恍惚惚，还跟发小一起做了个网站。哈哈，帮他打个广告金果庄园

一个字形容我的16年的生活 我选择 “浪“
16年，
看了大海，圆了演唱会梦，去了香港开开眼，拍到了真人詹姆斯and库里 满足。
业余爱好买了个尤克里里，到现在还只是会弹小星星
参加了发小、初中好友的婚礼 祝福。
买的一大推非技术类的书 就只重温完了《三体》（大学看的是残缺的pdf）
幸好幸好 技术类的书痛定思痛 没白浪费。

一个字形容我的16年的个人博客 我选择 “油“
当初开博客初心就没想把它作为单一的技术类笔记，所以大部分记得都是这一年的生活流水账，技术类博客写的也是零零散散。 明年的想法就是能多写写关于技术的，至少也有个技术分类集，最近在看java8 in action书，项目都已经上jdk1.8，我们还傻傻的用7和7以前的写法，惭愧。希望能写这书读后感，尽量能帮助我们的代码重构。

新的一年计划愿望
照搬15年的 然后加上一个,23333

  • 事业更上一层楼
  • 攒钱
  • 家人、我爱的人和爱我的人都平安健康幸福快乐
  • 和小葵好好的
  • 所有的愿望都成真
  • 减肥
