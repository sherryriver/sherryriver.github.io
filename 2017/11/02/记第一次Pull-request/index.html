记第一次Pull request

2017.11.02 sherryriver 流水账技术  pv:
  • ps：2017-11-21 更新
    一早起来发现邮件，过了这么久 第一次PR终于被作者采纳了 哈哈。

玩Github这么久了，才第一次向开源项目贡献代码 mark一下流程。

一

在github上找到自己想贡献代码的项目地址，点击fork

二

git clone 【你的仓库下fork原项目的地址】

三

git remote add upstream 【原项目仓库的地址】

四

修改文件
git add 【文件】

git commit -m “【备注】”

五

git push origin 【拉取的分支】
这一步完成后可以发现本地修改的已经push上去了

六

然后点击 new pull request 。如图片顺序,然后写上自己 pull request 的理由,成不成功 就看原作者同意否啦
