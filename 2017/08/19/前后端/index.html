【流水账】前后端接口交互传输记录

2017.08.19 sherryriver  pv:

博客荒废了好久了，最近忙成狗，抽了时间记录下。谈一下前后端交互传输一些规则，当然还是Java为后端spring框架。

前端常见的POST提交数据方式

Json格式
application/json; charset=utf-8
请求格式 {“title”:”test”,”sub”:[1,2,3]}

后端接收数据
public String add(@RequestBody PaymentChannelModel paymentChannelModel)

表单提交：
application/x-www-form-urlencoded;charset=utf-8
请求格式 name1=value1&name2=value2

后端接收数据
public String add(PaymentChannelModel paymentChannelModel)

public String add(@RequestParam Map map)

表单提交需要上传文件时
multipart/form-data 表单上传文件
