域名购买推荐指南

2017.10.27 sherryriver 流水账技术  pv:

宁愿吃土也要花钱

国内:

阿里万网 https://wanwang.aliyun.com/
大厂靠背安全放心，可选域名也多

易名中国 https://www.ename.com/
老牌域名商了

美橙互联 https://www.cndns.com/
通过同事介绍才知道这个注册商（原谅我的浅薄）
听说价格有优势

国外:

godaddy 俗称狗爹
算是全球最大的域名注册商了吧 也是我一直在用的注册商
一大优点 支持支付宝支付 哈哈

namecheap https://www.namecheap.com/
字如其名 优惠多多

Emoji域名 https://xn--i-7iq.ws/
特点：表情域名注册
看看就好

namesilo https://www.namesilo.com/
声称提供永久的免费域名隐私保护
看网友评价都还不错，下次可以试试

我一般是查域名的时候用国内的，响应快。个人观点，自己使用的话不推荐在国内注册
emmmmm 你懂的 受国内政策监管 不一定属于你自己的域名还有com域名需要备案才能使用，商用的话还是老老实实选国内吧。

如果在国外购买后一大头痛的问题就是访问速度，推荐使用dnspod解析，已使用速度杠杠的。

最后甩个吃土党福利
https://www.domcomp.com/
各大域名注册商的域名价格对比
妈妈再也不用担心我选域名纠结了

花钱？不存在的

一些大厂提供的二级域名使用
Github 全球最大的同性交友平台 哈哈
gitosc 码云 中国版的Github
coding 码市

局限就是只能在提供下做静态页面
如果只是官网的宣传推广的话够用了
